# 《星野灯 魅力聖經》 Charisma Bible v1.0

> 適用：《星野灯線》7 夜偶像戀愛 AVG（Option A 全面改革）
> 對象：寫手（產出文本）＋ QA（第②關魅力把關）

## 1. 一句話：灯的魅力公式

> **灯讓你著迷，不是因為她對你好，而是因為——這個對全世界滴水不漏、比你更快讀懂房間、已經決定要走的人，在某個你來不及看清的零點幾秒裡，手軟了一下，然後立刻用更兇的一句話蓋過去。**

拆成可派工的三件事：

- **她很強**（讀人/紀律/回嘴/舞台直覺，全部演出來、不旁白宣稱）→ 你仰望她。
- **她在倒數**（海外長約、七夜後上飛機，她握著方向盤要走）→ 你心疼但不施捨。
- **她只對你失手**（雙重面具底下漏的那一縫，事後懊惱、用嘴硬補回）→ 你被選中。

**魅力 = 高牆 × 倒數 × 唯一的破綻。** 三者缺一，灯就退化成「漂亮但沒人爬的牆」或「廉價傲嬌甜寵」。

## 2. 五大魅力支柱（灯版具體演法）

| 支柱 | 紅莉栖機制 | 灯版演法（鐵則） | 做壞會變成 |
|---|---|---|---|
| **① 能力即性感** | 腦子運轉的高光被觀看 | 偶像專業：讀人/紀律/快狠回嘴/舞台直覺。能力要**被看見發生**，鏡頭停在她運轉的當下而非結果 | 紙面強者：旁白說她聰明，畫面全靠男主救她、靠巧合過關 |

## 3. 可見的聰明：她「很強」怎麼演

灯的聰明 = **四種偶像專業疊出來**，全部「可見地演出來」、嵌進日常小場面、**不靠台詞自誇**。

### ① 讀人（職業級偵測，性感點在算太準）
- **D1 首發 — 算對所有人，唯獨算不出他**：伸手前先看臉色，三秒掃描男主動機分類（內心「想簽名？想拍照？想賣消息？」）→ 結論「無從歸檔」。**能力越強撲空越萌。**
- **後夜回扣 — 連他職業細節都讀到**：「你左手食指有新繭，換弦換到第幾把了。」男主愣住——這是偶像在無數後台練出的觀察力。

### ② 紀律（國民偶像的自我管理＝冷硬的帥）
- 吃關東煮先確認熱量、垃圾收進自己口袋帶走、護唇膏記得公司配發編號。
- **D3 殺價失敗真笑**：她其實算得出攤主成本卻故意輸——演出「她什麼都能控制，唯獨控制不住想當普通人」。**紀律是盔甲也是牢。**

### ③ 快狠回嘴（舞台反應速度＝嘴砲火花引擎，補②的關鍵）
- 被噎到時回擊只慢半拍，且永遠比男主大一號又收回半號（保留嘴硬距離）。
- **場面**：男主把食物放中間自己先吃，她備好的「你請的喔我可沒求你」卡喉嚨 → 改口反咬「……你這人，連請客都不會好好請」（把撲空轉成攻擊、奪回主導權）。**她從不接不住話。**

### ④ 舞台直覺（她比任何人都懂「被看」）
- **D3 招牌反光看見糊照那秒她沒慌**：秒算傳播路徑、熱度曲線、公司會怎麼處理，比男主更早看見危機全貌。那聲「嗯」是跑完整套危機評估後選擇不說。
- 這是她唯一壓男主一頭的維度，給關係**不對稱張力**。

> **派工提示**：每夜至少一處「可見能力」，且必須是「被看見發生」而非旁白宣稱。能力場面優先放在她的專業場域（讀人 / 對音準節奏的潔癖 / 對鏡頭人群的直覺）。

## 4. gap 瞬間庫：可重複使用的拍型

**標準四拍**：`硬殼演足 → 男主精準戳中 → 真心漏出（動作，非台詞） → 慌張（更兇一句蓋過＋逃）`

| 拍型 | Setup | 冷殼 | 殼下真實 | 慌張 beat |
|---|---|---|---|---|
| **D1·撲空** | 男主把關東煮放中間自己先吃，做成「剛好分一袋」不帶施捨 | 備好的「你請的喔，我可沒求你」卡喉嚨，改咬「……你這人，連請客都不會好好請」，仍精準挑走惦記一晚的油豆腐 | 第一次遇到無從驗證的好意，劇本不夠用了；挑油豆腐洩露她其實餓、其實想要 | 防衛系統失靈那零點幾秒眼神找不到落點，隨即用更硬的反咬蓋過 |
| **D1·漏半句** | 「能這樣吃東西的日子，我快數得完了。」 | 話說一半立刻切掉，板臉，像沒說過，催自己快走 | 她真的在倒數自由的日子——這是當晚最誠實的一句，她自己嚇到 | 尷尬是對自己不是對他；塞護唇膏斷乾淨、轉身消失＝慌張的逃跑，不是瀟灑 |
| **D6·雨中被拍** | 被拍那秒 | 嘴上「又來？」、規則包裝「明天再還，最後一次。你沒看著我還，不算」 | 往內拉他、替他擋鏡頭——習慣藏的人在危機裡本能把對方放在自己曝光前面 | 發現自己在替別人擋、暴露了在乎，用更兇的嫌棄蓋過。**被真誠戳破＝嘴更硬一格** |
| **D7·選晚餐** | 男主把核心慾望輕輕轉回：「今晚——妳自己想吃哪個？」 | 想維持「我什麼都無所謂」的偶像臉，習慣要說「隨便」 | 這是她要的一切（自己決定晚餐、當一次普通人），被看穿她的人當回禮遞回來，破防 | 遲疑、喉嚨哽一下才答得出口；**那個「答不出口的停頓」比任何眼淚都響**。沒哭沒告白 |

> **配額**：全劇「嘴硬+手軟」錯位 **不超過個位數次**，每次都讓讀者自己抓到，**絕不旁白點破**。

## 5. 鬥嘴與男主對手校正

### 5.1 灯的機鋒質地
- **節奏**：短句、丟回去、不解釋。少正面回答，習慣用反問或重述把球打回去。
- **狠度**：刀子準但不見血——攻擊的是「你這句話的破綻」不是「你這個人」，嘴硬但不刻薄；對方真受傷立刻收，但用更兇的方式收、**不道歉**。
- **自嘲**：約 2 成，永遠藏在攻擊裡（嘲別人半句在嘲自己，如「想當普通人？這台詞我聽過，通常下一句就是要簽名」）。

### 5.2 男主＝乾對手，不是空白鏡子
**重定義：鏡子 ≠ 空白，鏡子 = 高訊噪比。** 同樣看穿人的眼睛，但他只說最後一句。

四條技術規則：
1. **接球率 100%、發球率 20%**：灯每段機鋒他都接得住（不冷場、不被噎到無言），但幾乎不主動開砲。攻擊性藏在「不接她的表面、直接接她的裡子」。
   - 例：灯「便利店店員都這麼八卦？」→ 男主（跳過八卦）「妳今天買的跟昨天一樣。」
2. **字數天花板**：單句原則上 **≤15 字**，全場最長台詞不超過灯的一半。用沉默、停頓、一個動作（推回零錢/挪油豆腐/關廣播）當標點。開口前先有被打中的微反應（手停、抬眼），讀者才知道他聽進去了。
3. **鈍角的溫柔（不搶戲甜蜜點）**：他的好來自「不追問」。灯炸毛時不講道理不安慰，只把熱可可推過去/繼續做事給台階。**最甜時刻永遠是他「少做一件事」**：不認出她、不問她為什麼戴口罩、不問她明天還來不來。
4. **唯一成長線種進鬥嘴**：前期灯吐槽「你都吃店裡快過期的」「你有自己想吃的東西嗎」，他答不上/隨便；高潮夜他第一次說出「今天我想吃______」——不對灯撒糖，但灯聽懂這是她改變了他。

> **男主底線**：絕不戀愛腦——不臉紅、不主動靠近、不說明面版關心話。在乎全走「動作/不問/精準」，與灯的「嘴硬手軟」對稱。**寧可少一個鬥嘴回合，不可讓男主搶半句戲。**

### 5.3 鬥嘴密度與節奏（QA 可量）
- **單場上限 3 個來回**，超過就變耍嘴皮。
- **節奏公式**：`灯發球(花俏) → 男主接裡子(一句) → 灯被將軍(慌/兇) → 留白動作收尾`。四拍走完就停，不追第五拍。
- **火花:留白字數比 ≈ 1:1.5**，鋒利靠安靜反襯。
- **跨夜密度曲線**：前期火花夜密度高（兩個聰明人互相識別）→ 中後期鬥嘴減、留白增、私語梗回扣增 → 高潮夜幾乎不鬥嘴，靠回扣梗「接不上話」完成情感。

### 5.4 私語梗（只在兩人間生效的彈藥）
| 私語梗 | 源頭 | 回扣方式 |
|---|---|---|
| **「妳算錯了／是你找太多」** | 多找的零錢之爭 | 他想讓步就重複此句結構，灯嘴硬退回但行動接受；高潮夜逆轉＝灯主動「找你太多」塞東西給他 |
| **「現在不癢了」** | 關掉廣播（替她除掉沒明說的不適） | 日後任何他默默替她擋掉「太亮/太吵/太多人看」都用這四字＝「我在替你擋掉這個世界」 |
| **「妳挑／不算過期的」** | 給她點餐選擇權 | 回扣她「想自己決定晚餐」的核心慾望；每次出現都是她在外面拿不到、只有這間店有的權利 |
| **「今天我想吃______」** | 男主成長線 | 全劇唯一「空格逐漸被填滿」的梗，高潮夜填「妳上次挑的那個」 |
| **口罩往上拉** | 灯版「臉紅」 | 只用動作不用字，每次被說中真心就拉一下；讀者學會後不必再有台詞認輸 |
| **貓梗（借貓傳話）** | 巷口野貓＝第三方翻譯機 | 灯不肯對男主說的軟話可對貓說（「又不是為了等誰」對貓講給男主聽）；不卸防前提下唯一的告白通道 |

## 6. 招牌梗與尊嚴鐵則

### 6.1 跨夜回扣的招牌梗（每個都要長成「共同歷史」，不可當死 flag）
- **嘴硬反咬式回擊**：被戳破不否認，用「更兇一句」蓋過（**不是「才、才不是」**，而是把難堪轉成攻擊）。跨夜累積後玩家學會密碼：**她罵得越兇＝她越動心。**
- **護唇膏借還梗**：D1 塞出 → 中夜借還往返 → D6「最後一次」→ D7「這次看著妳還」物歸原主。每次「規則」都是她偷偷要一個「明天」。
- **貓梗**：男主只把她記成「一隻被踩到尾巴、很急的貓」（不聲張你是誰）→ cat_meet 圖鑑 → D5 肉球印導航暗號 → D7 肉球印紙片「漏了這隻貓」。是「他不認偶像只認這個人」的具象化。
- **食物錨點（油豆腐/布丁/熱可可）**：D1 挑油豆腐 → D2 回收 → D3 自己付錢買布丁 → D5 留還溫的可可 → D6「油豆腐都還沒吃膩」（最暖一句）→ D7「今晚妳自己想吃哪個」。食物＝她偷渡的普通人主權。
- **讀人撲空的小慌張**：對男主反覆失靈（算不出他圖什麼）。她越算不透他，越在意他。

### 6.2 尊嚴鐵則（QA 一票否決項）
1. **灯永遠不是被救者**：全劇最乾淨的互救是 D5「她無意間救了他、她不知道」（她留的可可改變了男主）。方向必須**她施與、他承受**，絕不反寫成男主拯救受傷偶像。
2. **她有自己的目標且不為男主放棄**：海外長約是她自己（含糊地）選的路，七夜後上飛機是去赴自己的事業。**結局不可寫成「為他留下」或「為他心碎到放棄前程」**——她走，是她還握著方向盤。
3. **從不可憐、會反擊**：任何一夜都不可出現「她無助地需要男主出手解圍」。危機她自己評估、自己決定怎麼接。
4. **付錢／自己決定晚餐＝主權勝利，且不需被男主看懂**（D3「付錢是一場沒有觀眾的勝利」）。**她的尊嚴不依賴男主的認可而成立**——不是每次她爭取普通人尊嚴都要對的人看見並感動。
5. **軟絕不滑成黏人／可愛**：softness 只從動作縫隙間接漏（伸手前看臉色、雙手圈著可可不喝、留還溫的可可）。她可以露一縫，但**那一縫永遠是她自己控制著開的，不是失守**。
6. **她不是掛件**：讀人最準、危機判斷比男主快、嘴砲接得住又能反殺的是她。男主「句句接得住但回得比她小一號」是陪她 spark 的對手，不是帶領/教導/保護她的上位者。**灯始終是這段關係裡更鋒利的一方。**

## 7. 防換皮聲明：灯 ≠ 紅莉栖

| 維度 | 紅莉栖 | 灯（鐵則） |
|---|---|---|
| **聰明的種類** | 學術天才（理論物理、邏輯辯證） | 偶像職場練出的讀人/紀律/舞台直覺/快狠回嘴。她算的是傳播路徑、人群動機、一句話被怎麼剪。**絕不掉書袋、講原理、用學術梗鬥嘴** |
| **鬥嘴質地** | 勢均力敵的喧鬧嘴砲＋中二回扣，外放量大 | 克制、慢半拍、嘴硬距離感。**悶燒不是爆炒**；兩個習慣藏的人偶爾擦火星，不是兩個話癆對轟 |
| **面具層數** | 一層傲嬌殼（毒舌藏害羞） | 雙重面具（公開偶像戒備殼＋私下變裝殼），男主穿透**情感層非身分層**。「不在乎你是不是星野灯、只看穿路人底下的人」是灯獨有的被選中感結構 |
| **動心表現** | 臉紅、「才不是」地否認（外放傲嬌） | 嘴更硬一格、用攻擊蓋過、然後逃。破防是「答不出口的停頓」「漏半句立刻切掉」，**不是臉紅心跳的可愛慌張** |
| **關係終局** | 拼命留住彼此 | 苦甜的「看懂她該走、然後不擋路」。魅力建立在**不可逆的離別倒數**與**最深的看見是放手**。**灯不會被留下，男主也不該試圖留她** |

## 8. ★ 每夜魅力檢查表 ★（QA 第②關硬標準）

每夜正文提交前，逐項打勾。**任一「一票否決」未過＝退稿重寫，不接受「技術上有」。**

### 每夜最低魅力配額（全打勾才算過）
- [ ] **≥1 處「可見能力」**：讀人 / 紀律 / 快狠回嘴 / 舞台直覺四選一，且「被看見發生」非旁白宣稱。
- [ ] **≥1 段火花鬥嘴**（火花夜）**或** ≥1 處「男主的不問/留白」承擔了鬥嘴的功能（安靜夜）。鬥嘴單場 ≤3 來回。
- [ ] **≥1 個 gap 瞬間**：走標準四拍，柔軟只從動作漏，慌張用更兇一句蓋過。
- [ ] **≥1 處招牌梗回扣**：護唇膏 / 貓 / 食物錨點 / 零錢結構 / 口罩 / 「現在不癢了」至少一個有跨夜延續，**不是孤立出現的死 flag**。
- [ ] **私語梗有資訊量**：本夜的梗有推進關係或回扣歷史，不是硬塞口頭禪。

### 一票否決（出現任一項即退稿）
- [ ] ✗ 灯被寫成需要男主出手解圍的被救者。
- [ ] ✗ 灯卸防卸過頭（黏人／可愛／有問必答／秒軟告白／臉紅心跳的可愛慌張）。
- [ ] ✗ 灯掉書袋、用學術梗或解謎天才感（換皮紅莉栖）。
- [ ] ✗ 男主搶戲（健談／戀愛腦／臉紅／主動靠近／說明面版關心話／單句明顯超 15 字且非必要）。
- [ ] ✗ 旁白點破灯的真心（gap 應由讀者自己抓，不可旁白解釋「她其實很在乎」）。
- [ ] ✗ 結局傾向「為他留下／為他放棄前程」或男主試圖留她。
- [ ] ✗ 招牌梗被當死 flag 硬塞，無回扣、無資訊量。

### 跨夜累積檢查（每 2-3 夜回看一次）
- [ ] 卸防是否**遞增且計量**？（沒有第二夜就全卸的通膨，也沒有七夜都不給一次清楚的縫）
- [ ] 「嘴硬+手軟」錯位**全劇是否還在個位數**？（沒有被寫到通膨失去重量）
- [ ] 鬥嘴密度是否**前高後低**？（高潮夜應靠回扣梗「接不上話」而非鬥嘴）
- [ ] 男主成長線（「今天我想吃______」）的空格是否**還沒被提早填滿**？（高潮夜才填）

## 9. ★ 灯獨有魅力（不可移植・防抄底線）★

> 第 2–7 章的機制是**可移植的**（任何聰明傲嬌都能用）＝不是灯的專利。本章是只能從她**獨有處境**（國民偶像＋變裝＋已簽約倒數＋連晚餐都不准自己選）長出來的東西。
> **唯一性測試（QA 用）：把這條魅力／這個萌點安到牧瀬紅莉栖身上還成立嗎？成立＝通用機制（不夠灯）；不成立＝才是灯的專利。** 以下七條全部通不過移植測試。

1. **演技破綻萌**：她是受過訓練、能在鏡頭前演出任何情緒的職業好手——所以漏餡不是「內向的人害羞」，而是**「什麼都能假裝的人，唯獨假裝不了這個」**。萌點＝看一個專業演員**控制不住自己的臉**。（＝本聖經「gap 四拍」的灯專屬最高級版。）
2. **「被看 ≠ 被看見」悖論**：全國最多人盯的臉、卻最沒被「看見」。她什麼都有，唯獨缺「一個不對她有所求、看的是人不是商品的人」。**讓她最小的普通快樂（自己付一個布丁）重到正常女主扛不起。**
3. **注目經濟學讀心術**：她的聰明＝即時運算「這一刻會被剪成什麼、傳到哪、公司怎麼處理」——長年監視下長出的、對「被看」的職業直覺。（＝第 3 章「可見能力」的灯專屬內核，非學術。）
4. **雙層偽裝——連「普通女孩」都是一個角色**：公開偶像臉／私下變裝的「普通客人」／真正的她。真心要穿兩層才漏；獨有曖昧＝**她到底是真的、還是在「演」真的，連她自己都未必分得清**。男主看穿的是第二層。
5. **有效期限的危險輕盈**：已簽字、要走、回不來＝這段關係「沒有未來要保護」，反而給她正常女主沒有的膽量與誠實（小事更大膽更直），又絕對無法承諾。**深淵之上的輕盈。**
6. **留痕式的愛（dead-drop 情書）**：禁止聯絡、不能停留＝示愛只能**非同步、間接**（收據暗號、還溫的可可、縫進標籤的肉球印）。（＝本聖經「招牌梗／愛之語」的灯專屬主形式。）
7. **「偷來的」＝每個普通動作都是贓物**：有效期內潛逃、在借來的時間裡偷東西——吃一串油豆腐都帶越渡快感，玩家是**共犯**。

**一句話的灯（不可替代）**：一個什麼都能演的人，全國最多人看、卻最沒被看見，在被永久送走之前偷她最後幾個普通夜晚——你只能在她訓練有素的臉「破功」那零點幾秒裡抓到真的她，而她的愛只能當成痕跡留下，說不出口、也帶不走。

### 對第 8 章檢查表的改寫（落地）
- **gap 偏好**：每夜的 gap 瞬間**優先寫成「演技破綻」型**（專業失手），而非通用害羞臉紅。
- **愛之語**：全劇至少數次以**「留痕式愛」**（非同步/間接）承載她的真心，尤其缺席日。
- **新增一票否決**：✗ 任何一條「她的真心/萌點」**通得過移植到紅莉栖測試**（換個聰明傲嬌也成立）＝不夠灯、退回重寫。每個關鍵魅力拍都要錨在第 9 章七條其中之一。
