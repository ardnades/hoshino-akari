# 《星野灯線》七夜角色推演報告

> showrunner 整合稿。對象：spec/23 重構藍圖 vs 七夜「角色入戲推演＋場面編織＋劇本控制員稽核」。立場：用角色的真實行為檢驗藍圖，凡角色「拒絕照藍圖走」之處，預設是**藍圖該改**，不是角色該改。

---

## 1. 總結論

**角色實在感：很高，高到危險。** 這七夜推演最值得信任的不是「場面好看」，而是兩個主角的 `refusal`（拒絕清單）幾乎逐條成立、且彼此咬合。灯與男主都是「習慣藏的人」，推演把他們放進七個情境後，反覆長出同一種質地：**雙向克制、互相看穿、誰都不戳破，玩家是唯一握有全貌的人。** 這正是 spec/23 主題鐵律（「最深的看見，是看懂她該走、然後不擋路」）能不能成立的試金石——而它通過了。

**但最重要的訊號不在「角色守住了藍圖」，而在「角色拒絕照藍圖走、且拒絕得比藍圖更對」的那幾處。** 它們是藍圖該改的高優先警報：

1. **【最強訊號】男主從頭到尾不該「認出星野灯」，只該「看穿這個人」。** 藍圖（與 D1/D7 推演）一度把 `fs_male_recognizes`（他第一夜就認得國民偶像、按住七夜不說）當核心底牌。但 landed day7.js 的 `seen_through_flag` 回扣（「你從第一天，就看穿我」）說的是**他看穿她在逞強、在餓、在演**，不是認出藝人身分。**情感層的「懂這個人」比身分層的「認出偶像」高級得多，也更貼 CLAUDE.md「softness 漸進、間接」。** 角色的真實行為（他遞而不說、不翻牌、不消費）天然指向「他根本不在乎她是不是星野灯」——這比「他知道是星野灯卻忍住」更動人。**藍圖該把 fs_male_recognizes 從身分認知降級成情感認知。**

2. **【第二強】「驗證撲空 / 嘴硬卡喉嚨」是全劇最強的單拍，藍圖沒寫。** D1 灯整套防衛系統是為「驗證對方圖什麼」設計，男主把食物放中間、自己先吃，做成「剛好分一袋」，讓她準備好的「你請的喔，我可沒求你」對不上場面、卡在喉嚨。**一個活在「沒人對她無所求」世界裡的人，第一次遇到無從驗證的好意——她的劇本不夠用了。** 這是角色自己長出來的、比任何軟化都真實的瞬間。藍圖只把它當 emergentMoment，應升格為 D1 必演拍。

3. **【系統級】heat / 班機日期 / 多個伏筆是「藍圖敘事引擎」，但引擎層（meta.js judge）根本不追蹤。** 推演反覆把 heat≈85、12號班機、票不可退當主引擎，但 judge 偽碼讀的是 affection/awareness/regret/warmth/stance＋flags，**沒有 heat 變數**，且 `pushed_to_choose`/`let_her_go`/`gambled_on_being_seen` 在 landed 文本裡幾乎沒被設過。**heat 是氛圍，不是機制；班機是情緒倒數，不是資訊倒數。** 藍圖必須誠實標記哪些是「可結算的軸」、哪些是「世界觀調味」。

4. **【最該警惕】藍圖（神社D5・洗衣店D4）與 landed 正文（停車場D5・販賣機/雨棚D6）是兩套不同的劇本。** 第 4、5、6 夜的推演 JSON 自己抓到了這個落差並改判以 landed 為準——這是對的。**任何把這份推演當「已實裝」來派工的人都會誤判。**

## 2. 逐夜精華

### D1 後巷相遇 ‧ 好看度：強
- **真正發生**：撞帽揭臉零點幾秒→空口袋→他把關東煮放中間自己先吃（不「給」）→她挑那塊惦記一晚的油豆腐→她漏半句「能這樣吃東西的日子，我快數得完了」立刻切掉→塞護唇膏斷乾淨→消失。
- **最動人浮現**：「**伸手前先看臉色**」。一個被千萬人崇拜的人，拿一塊十幾塊的油豆腐還要先確認能不能拿——沒寫進任何台詞，卻比那句「普通」洩露得更徹底。
- **三方對撞最佳解**：護唇膏三讀法零交集（她＝斷乾淨／他＝中性收據不賦予意義／世界＝連這支都是公司配發、借來的）。**不和解本身就是這一夜苦甜的結晶**，玩家是唯一看見三個版本的人。
- **缺口**：貓梗（「我只看到一隻很急的貓」是 D1 副標、cat_meet 圖鑑解鎖句）零回扣，斷了到 D7 肉球印的源頭。必補一次「貓」的命名（建議男主內心，且與「不聲張你是誰」咬合：他刻意只把她記成一隻貓）。

### D2 末班電車 ‧ 好看度：強
- **真正發生**：她並排站（同時是靠近與逃生）→他扶杆挪半步又收回（給支撐不給手）→官方「重大企劃」推播，他用力一點把螢幕扣下→她讀成「他沒興趣」→鄰節車廂飄來「星野灯要宣布大事了」→她「別跟下來」、他本來就要先轉身。
- **最動人浮現**：「**兩種動機撞出同一結果**」——他扣手機是「替她關鏡頭」，她讀成「他沒興趣／不消費我」，同一動作在兩人腦中是兩部電影，卻都讓她安全。**因為誤會而剛好對的溫柔**，藍圖寫不出，只能從行為長出來。
- **三方最佳解**：她從倒影回望的半眼 vs 他先轉身錯過——**體貼導致的擦肩**，命中主題。
- **缺口**：油豆腐錨點（台帳要求 D2 回收）在末班車版本完全缺席（landed day2 其實是油豆腐夜，藍圖的末班車是另一套）；男主「下個月我也得去一個地方」側寫台詞未出現於對白。

### D3 章魚燒攤 ‧ 好看度：強
- **真正發生**：她堅持自己點、自己付（空口袋舊刺回收）→被燙嘴硬蓋過→殺價失敗真笑→大叔半認出又自我否定→她在招牌反光看見「#深夜目擊 後巷糊照」（heat 對玩家點亮）。
- **最動人浮現**：「**『嗯』的雙重曝光**」。她僵住看糊照，男主只回一個字——對她是隨口附和，對玩家是「我也看見了、我選擇不放大」的全部重量。**同一個字在兩人耳裡是兩件事。**
- **三方最佳解**：喜劇／見證／素材三層定義同框為真。大叔那句「大明星哪會被燙到還嘶嘶嘶」——**世界最大的溫柔與最大的殘忍是同一句話**：她越像人越安全，也越接近被當成新聞素材。
- **錨點落點爭議**：焦糖布丁。已查證 landed day3 確實有「自己數硬幣付錢買布丁＋cg:pudding」——台帳「落點懸空」是**誤報**。

### D4 中點翻牌 ‧ 好看度：強（但 landed≠藍圖）
- **藍圖版真正發生**（洗衣店玻璃盒）：她拿下帽子（甜最高點）與路人舉手機（咬一口）幾乎同框→經理人「12號、票改不了」訊息→熱可可她雙手圈著不喝→他「我也要走了」假輕鬆、她聽穿不拆→襪子放回袋子上面。
- **最動人浮現**：「**保護一件即將出賣她的衣服**」——他把烘衣溫度調低（「棉的，高溫會縮」），保護的恰恰是那件會被新聞認得的淡藍帽T。
- **嚴重矛盾**：landed day4.js 仍是**舊版停車場販賣機**，不是藍圖的洗衣店。採藍圖＝需改正文＋新 bg＋STORY_BIBLE，屬重構工程。

### D5 缺席日 ‧ 好看度：強
- **landed 真正發生**（停車場，非神社）：零接觸、零文字、純物件對位——她留還溫的可可＋布丁收據（肉球印導航）；他回未開封的可可立同位、蓋好油豆腐盒給她拿、護唇膏收進口袋最深。
- **最動人浮現**：「**她無意間救了他**」。她留可可純粹替自己止血，完全不知道這罐還溫的可可在男主心裡擋下了他對自己人生的認輸。**整夜唯一真正改變了什麼的事件，發生在缺席者毫不知情的情況下。**
- **三方最佳解**：「站住／回頭／然後走」的反英雄三拍——他看見自己留的東西還亮在曝光現場，沒衝回去收。「看見了卻只能站著」是他「救不了她」最具體的一幀。
- **嚴重矛盾**：藍圖（神社＋抽籤＋經理人焊縫隙）與 landed（停車場＋收據解謎）根本不符。**神社藍圖視為作廢歷史版本。**

### D6 雨中 ‧ 好看度：強（但 landed 把名場面演成過場）
- **landed 真正發生**：販賣機照片傳開→「不要回頭」相遇→並肩淋雨→「油豆腐都還沒吃膩」（七夜最暖一句）→塞回護唇膏「明天再還，最後一次」→「又來?」接住。
- **最動人浮現（被 landed 漏掉的金礦）**：「**雙向擋鏡頭的笨拙錯位**」——被拍那秒她往內拉他、他往外讓半步，方向相反疊在一起。**兩個習慣藏的人在危機裡不約而同把對方放在自己曝光前面，卻誰都沒發現對方在做一樣的事。** landed 只演了單向（她慢半步），這一拍該補。
- **三方最佳解**：「**一千隻溫柔的手**」——留言區沒人罵她，全是「擔心那個男的會不會有事」，而正是這份善意把那個男的判了死刑。傷害全部來自溫柔、沒有施害者可以被恨。

### D7 命運結算 ‧ 好看度：強
- **landed 真正發生**：護唇膏「這次看著妳還」物歸原主→「自由是什麼味道……很貴」（全劇主題句）→未來想像半句「給——」被「灯，時間。」準時切斷→結尾肉球印「她把該收的都收走了，就漏了這隻貓」。
- **最動人浮現**：「**今晚——妳自己想吃哪個?**」——男主把灯整條線的核心慾望（自己決定晚餐）當回禮輕輕轉回給她。抽象的「不擋路」變成一個會痛的小動作。
- **三方最佳解**：世界對這場戲一概不知、也不在乎——它只把它讀成「風險變數待清零」。**兩人經歷的是一生一次的結算，世界經歷的只是一次準時的交件。** 這個量級落差就是整夜苦甜的引擎。
- **缺口**：男主第七夜是純功能性接球者，藍圖給他的「我明天也得去個不太想去的地方」一句沒用上，他的放手因此缺少代價。

## 4. 矛盾與出戲清單（按嚴重度）

**高**
1. **藍圖 vs landed 場景三夜性對不上**（D4 洗衣店/D5 神社/D6 雨中豐富版 ≠ landed 停車場販賣機/雨棚簡版）。修法：明確定案以 landed 為唯一真實；神社/洗衣店藍圖標作廢；後續派工一律引用 day*.js，勿據藍圖回改加素材。
2. **fs_male_recognizes 身分認知矛盾**（藍圖建立在「他認得星野灯」，landed 只有「看穿這個人」）。修法：定案情感層、修正藍圖（推薦）。
3. **D7 選擇結構 vs judge 偽碼脫鉤**：`pushed_to_choose`/`let_her_go`/`gambled_on_being_seen`/`heat≥85` 是 judge 命脈，但 landed 文本幾乎沒設這些 flag、引擎不追 heat。修法：開寫前先在一處（建議 spec/23 §heat 矩陣＋judge 偽碼）鎖死哪些 flag 真被讀、heat 是否入結算；勿讓三份文件各報一個數。

**中**
4. **D6 被拍那秒（名場面）演成過場**——雙向擋鏡頭錯位金礦被 landed 漏掉。修法：補 2-3 行旁白，男主先有極短「往旁邊讓」、與她的拉形成方向相反錯位，旁白只呈現不解釋（0 新 flag、純文本）。
5. **endings.js warm_true 開頭 vs day7.js 結尾三句逐字重複**（「路過便利商店沒停／想起 D5 熱可可/今天沒多買油豆腐」）。修法：endings 開頭改一句過渡（「那之後的幾天，我照常上下班。」）直接進電車訪談。
6. **D5 d5s4「兩塊都吃掉」flavor 支 vs S5「把油豆腐盒給她拿」狀態矛盾**（都吃掉了哪來一盒）。修法：S5 該句條件化，或把「都吃掉」支改成「只吃了自己那塊，第二塊還是留下了」。
7. **班機顯性倒數在 D5 缺席**（七夜主引擎，剩三夜卻無讀秒）。修法：D6 已沿用 SNS 兌現地點曝光；班機日期在 D7 補硬資訊即可。

**低**
8. **D6 同夜兩選項共用 almost_confession_flag**（布林，S2 觸發後 S4 無增量）。確認即可，非 bug（符合 spec/22 flag 真相鎖）。
9. **vending_machine_memory 死 flag 風險**：勿為對齊作廢藍圖新增無人讀取的 flag（呼應 pudding/cocoa_memory 教訓）；如需 D5 狀態請讀 affection/regret。
10. **糊照露臉鐵則邊界**：糊照素材只做「淡藍帽T背影＋模糊」，不做正臉/側臉；SNS 用既有 ui:sns 文字呈現，不出實圖。記入 assets 規格。

## 5. 角色實在感升級（應寫進正文/回填藍圖）

**灯（星野灯）**
- 「**伸手前先看臉色**」（D1）：升格為必演近景一拍。spec 診斷指出「鏡頭語言缺席＝廉價感根因」——這一拍是 0 新文本就能買回質感的地方。
- 「**驗證撲空 / 嘴硬卡喉嚨**」（D1）：正式寫進 D1 演出，而非只當 emergent。她的劇本不夠用，是「一個沒人對她無所求的世界」最痛的一格。
- 「**沒接第二句的『普通』**」（D1）：缺席的台詞比說出口的更響——沉默是她當晚最誠實的一句話。守住，別補嫌棄第二句。
- 「**付錢是一場沒有觀眾的勝利**」（D3）：男主不理解這筆錢對她的主權重量，刻意不讓他看懂。**確立全劇原則：不是每次越界都需要被那個對的人看見。** 比「他看懂了並感動」更苦更真。
- 「**規則包裝的撒嬌**」（D6/D7）：「明天再還，最後一次。你沒看著我還，不算」——她唯一付得起的撒嬌，用規則的形狀去要不敢開口的「明天」。landed 把卸防全壓在這上面（不加口罩戲）其實更貼 canon。

**男主**
- 「**給支撐不給手**」母題：D2 扶杆挪半步又收回 → D3 冰麥茶推兩公分 → D4 兩雙手各自找活幹（她圈可可、他配襪子，以避免伸過那一台洗衣機的距離）。**寫進 spec/21 演出層當男主標準語彙、跨夜複用。**
- 「**他認得這個人，不認偶像**」：把他的「懂」固定在情感層。他遞而不說、不戳破、不留電話、不問名字、不在退路放鉤子——全部指向「他根本不在乎她是不是星野灯」。
- 「**靠牆、不擋路、看著她走**」（landed D7「我站在雨棚下，靠牆的一側。不擋路。」）：landed 比藍圖的「擋光不擋路／備好熱飲零錢」更省更準。**藍圖該向 landed 看齊**，刪掉過度服務化細節。
- 「**身體比腦先記得她**」（D5 手自己多夾兩塊油豆腐而不自覺）：寫進角色聖經當男主「不言之愛」的定錨動作。
- 「**第七天也有不想去的地方**」：補回 D7（最高 CP 值），讓他的放手有代價、從工具人救成有重量的人。

**關係**
- **零交集的雙人默劇**：背對背各自保護對方、誰都沒承認（D3 都看見糊照都裝沒事；D6 都為對方擋鏡頭）。這是這條線的克制美學核心，後夜不可由任一方點破。
- **三層認知差疊在一句鬥嘴詞**（D6「又來?」）：灯以為藏得好／男主全懂／玩家看見男主懂。最深的看見藏在最不在乎的一句話裡。

## 6. 藍圖回饋

**角色行為證明藍圖「對」的地方（保留、勿動）：**
