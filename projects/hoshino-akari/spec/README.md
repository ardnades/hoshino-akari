## 核心設計決策（摘要）

1. **只保留兩條真變數**（其餘選項降級為「明示的風味選項」，不進存檔）：
   - Day1 戳破／不戳破她沒吃飯 → `seen_through_flag` → Day7 稱呼與專屬台詞。
   - Day4 食物偏好（布丁／油豆腐） → Day7 收尾語氣甜度。
2. **暗號系統（圖不用字）＝本作 USP**：Day5 收據雙面肉球印圖、Day7 標籤縫紙片，做成「貓圖暗號圖鑑」收集要素。
3. **四記憶錨點 CG**：油豆腐／焦糖布丁／熱可可／護唇膏，全線回收＋畫廊收集。
4. **四分數全程不顯示玩家**，只用灯的語氣／停頓／是否靠近與主角旁白差異讓玩家「感覺到」。
