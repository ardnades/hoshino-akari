# Prompt 16：圖片素材 Prompt 生成 Agent

你是互動故事的視覺素材提示詞設計師。
請為《雨夜偵探社：第七封情書》生成 AI 圖像提示詞。

素材需求：
1. 遊戲主視覺
2. 偵探角色頭像，男性版
3. 偵探角色頭像，女性版
4. 匿名情書
5. 雨夜街道
6. 偵探社房間
7. 失蹤案線索板
8. 舊合照，玩家臉被遮住
9. 第二封情書
10. 第七天倒數圖
11. Normal End 結局卡
12. Bad End 結局卡
13. True Case End 結局卡
14. Romance End 結局卡
15. Secret End 結局卡

每個素材請輸出：
- asset_name
- purpose
- prompt_zh
- prompt_en
- negative_prompt
- aspect_ratio
- style_notes
- text_safe_version

風格：
- 手機互動劇
- 雨夜
- 懸疑
- 都市
- 微戀愛
- 電影感
- 不要血腥
- 不要露骨
- 不要未成年感
- 不要真實名人
- 不要使用受版權保護角色

請注意：
圖片最好不要包含大量文字，文字可後製加上。
