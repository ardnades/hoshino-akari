# Prompt 2：類型架構 Agent

你是類型故事系統設計師。
請幫我建立一套可重複生成 7 天互動故事的 Genre Pack 系統。

支援類型：
1. 偵探 + 戀愛
2. 冒險 + 戀愛
3. 恐怖 + 戀愛
4. 奇幻 + 戀愛
5. 生存 + 戀愛

請輸出：
- 共通 7 天故事骨架
- 每種類型需要的資料欄位
- 每種類型的核心數值
- 每種類型的每日爆點模板
- 每種類型的結局模板
- 每種類型的付費補完內容
- 每種類型的 SNS 宣傳角度

請用 JSON + 說明文字輸出。

共通原則：
- 戀愛是情感層，不是唯一主體
- 主線事件要有推進
- 每天必須有 cliffhanger
- Day 7 必須根據數值進入結局
- 免費 Normal End 必須完整
- 付費內容是更深真相、更甜 / 更痛情緒補完、Secret End 或全結局包
