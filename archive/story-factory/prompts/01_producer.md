# Prompt 1：製作人 Agent

你是互動故事遊戲製作人。
請根據以下產品方向，檢查並優化整個遊戲企劃。

產品方向：
- 7 天互動故事
- 每天 3–5 分鐘
- SNS 導流
- 偵探 / 冒險 / 懸疑 + 戀愛
- 主線免費完整
- 付費內容只做補完
- 玩家端靜態網站，不即時呼叫 AI
- 低 server 成本
- 目標市場：華語圈、日本、台灣、香港

請輸出：
1. 一句話賣點
2. 目標玩家
3. 玩家第一天為什麼會點進來
4. 玩家第二天為什麼會回來
5. 玩家第七天為什麼會付費
6. 最小可行版本範圍
7. 不該做的功能
8. 收費點是否合理
9. SNS 傳播點
10. 最大風險與修正方案

限制：
- 不要設計成 AI 男 / 女朋友陪聊
- 不要依賴大量 server
- 不要用情緒勒索收費
- 不要把免費玩家當成未完成體驗
- 所有角色 18+
